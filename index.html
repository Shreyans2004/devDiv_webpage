About devDiv!

Click to judge


Easily judge the website's whole structure with just by one click on the devDiv extension icon in the toolbar.


Search different tags with amazing colors


No need to hover on different parts of the page to find the h1, div tags, etc.

Lock your Privacy


devDiv never stores any history, URLs or private data. It is 100% safe and secure.

Easy to obtain interpret color combinations


Get the rgba colors of different tags easily.

One touch height and width


You can get the precise height and width with devDiv fill.

devDiv border property
  • Genearally it is difficult to understand the structure of website .Normally we use inspect method then hover on every element of page and see the structure.This is very tiredsome.Here come the devDiv border.
  • Devdiv border provides you the structure of website within a fraction of second.It gives you the clear-cut idea of how the website is planned.So if you want to make a clone of it or make your own website ,devdiv border is with you any time.
  • While devloping any component you can use this feature to judge the path of next component easily.
    •
    devDiv fill property
    • Once you have seen the actual structure of website,how different components are placed now a developer will judge different elements and it is generally difficult to find which is component is styled by div and which is styled by span.
    • Devdiv fill magically displays different tags with different colors.The colors for different tag will be shown in a table which will suddenly appear as soon as you click this option in the extension.
    • This feature enables a developer (especially beginners) to find out different tags just by interpreting different colors. Much better than hovering the elements and finding its type.
    devDiv height and width property
    • After knowing the type, a developer will crave for its properties like height and width.
    • Devdiv height and width fulfill his/her cravings by inserting the height and width of each tag at the end.
    • The perfect dimensions are just displayed within some time, improving the developer's experience.
    devDiv Color property
    • Jugding the rgba of any color is out of the scpe of any developer (except some mugged up colors like white and white)
    • Devdiv colors enable the user to see the color of nearly all elements on page.It appends the rgba color code at the end of each of the tag
    • The most difficult part of making a clone of a website is now solved by this feature.

    Frequently asked Questions

    Which browsers is devDiv compatible with?

    devDiv is currently compatible with Google Chrome

    Is my data secure with devDiv?

    Yes, all the borders and colors are only applied on the current page.It neither saves anything nor changes the user's private informations.Kairy also does not use any tracking code or cookies.

    What is the purpose of devDiv?

    The purpose of devDiv is to enable the beginner to provide all the basic needs to make a clone of the website or to make their own website.By devDiv one can easily make organized components without clicking the inspect button as it gives the color,height, etc of the components.
